# Server mods — дополнения для выделенного сервера

Здесь лежат моды, которые **не входят в клиентскую сборку**, но полезны при
запуске выделенного сервера. Просто скопируй нужные `.jar` в `mods/` сервера.

> 💡 Если ты качаешь **`*-server.zip`** со страницы [Releases](../../releases) —
> эти моды **уже внутри**, ничего копировать не нужно.

| Мод | Лицензия | Зачем |
|---|---|---|
| `LuckPerms-NeoForge-5.4.140.jar` | MIT | Гибкие права и группы игроков. Без него — только ванильные OP-уровни. [Документация](https://luckperms.net/wiki/Home) |

## Что уже есть в самой сборке (ставить отдельно не надо)

Сборка изначально собрана с оглядкой на сервер, поэтому весь стек стабильности
уже внутри:

| Мод | Зачем |
|---|---|
| **FerriteCore** | Резко снижает потребление памяти (блокстейты, модели) |
| **ModernFix** | Ускоряет запуск, чинит утечки памяти |
| **Lithium** | Оптимизация серверной логики (ИИ, физика, тики) |
| **Clumps** | Склеивает шарики опыта — меньше сущностей |
| **Let Me Despawn** | Агрессивнее чистит лишних мобов |
| **Chunky** | Предгенерация чанков (см. ниже) |
| **Spark** | Профайлер: `/spark profiler start` для поиска лагов |
| **SimpleBackups** | Автобэкапы мира (раз в 2 часа, 10 копий, пустой сервер не бэкапит) |
| **PacketFixer** | Чинит разрывы соединения на больших пакетах (актуально для GT/AE2) |
| **FTB Chunks + FTB Teams** | Приваты территорий и команды |
| **Lootr** | Персональный лут в сундуках — не растаскивают друг у друга |
| **Simple Voice Chat** | Голосовой чат (порт UDP 24454) |

## Осознанно НЕ добавлено

Эти популярные «серверные» моды в этой сборке **вредны** — проверено:

- **ServerCore** — душит тик чанков под нагрузкой, из-за чего встают машины
  GregTech и контрапции Create;
- **Alternate Current** — переписывает логику редстоуна, конфликтует с MoreRed;
- **Noisium** — оптимизирует *ванильную* генерацию мира, а здесь генерация TFC.

## Предгенерация мира (сильно ускоряет игру)

Chunky уже в сборке. На запущенном сервере:

```
chunky radius 3000
chunky start
```

Прогресс — `chunky progress`. На 16 ядрах область радиусом 3000 блоков
генерируется примерно за полчаса-час и снимает почти все лаги от генерации.

## Рекомендации по железу

- **RAM**: 8 ГБ минимум, **16 ГБ** — комфортно (в `user_jvm_args.txt`
  серверной сборки уже прописаны 16 ГБ + флаги Aikar'а);
- **Java 21** обязательна;
- `allow-flight=true` в `server.properties` — **обязательно**, иначе корабли
  Create: Aeronautics и джетпаки GregTech будут кикать игроков за читы.
